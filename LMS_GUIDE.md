## 🟢 Moodle

### الخطوات:

#### 1. التحضير
```bash
# بناء الدورة
npm run build

# النسخ الناتجة ستكون في مجلد .output/public/
```

#### 2. رفع على Moodle
1. **تسجيل الدخول** إلى Moodle
2. اذهب إلى **الدورة المراد إضافة المحتوى إليها**
3. فعّل **وضع التحرير** (Edit Mode) - أعلى يسار الصفحة
4. اضغط **"إضافة عنصر أو نشاط"** (Add an Activity or Resource)

#### 3. اختيار طريقة الإضافة

**الطريقة الأولى: رفع ملف HTML**
1. اختر **"الملف"** (File)
2. أعطِ الملف اسماً: "درس الحوكمة الصحية"
3. قم بتحميل ملف `index.html` من المجلد `.output/public/`
4. اضغط **"حفظ وعرض"**

**الطريقة الثانية: استخدام SCORM (الأفضل)**
1. اختر **"محتوى SCORM"** (SCORM package)
2. انقر **"اختيار ملف"** وحمّل ملف `.zip` الذي يحتوي على الدورة
3. قم بالإعدادات:
   - ✅ **الدرجة الكاملة:** 100
   - ✅ **حد النجاح:** 60
   - ✅ **محاولات مسموحة:** 3
4. اضغط **"حفظ وعودة للدورة"**

#### 4. الإعدادات الإضافية (في Moodle)
```
التقييم والدرجات:
├── الدرجة الكاملة: 100 نقطة
├── درجة النجاح: 60 نقطة
├── محاولات مسموحة: 3
└── إظهار النتيجة: فوري

التتبع:
├── تتبع التقدم: مفعّل
├── علامات إكمال: مفعلة
└── التقارير: متقدمة
```

#### 5. التحقق
1. قم بتسجيل الخروج
2. ادخل كطالب وافتح الدورة
3. تأكد من عمل جميع المحتويات

## 🔵 Canvas

### الخطوات:

#### 1. التحضير
- بناء الدورة كما في Moodle
- إعداد ملف HTML جاهز

#### 2. الإضافة على Canvas
1. **تسجيل الدخول** إلى Canvas
2. اذهب إلى **الدورة**
3. اختر **"الموارد"** (Modules) أو **"الواجبات"** (Assignments)
4. اضغط **"إضافة عنصر"** (Add Item)

#### 3. خيارات الإضافة

**الخيار الأول: رابط خارجي**
1. اختر **"رابط خارجي"** (External URL)
2. أدخل الرابط: `https://your-server.com/governance-course`
3. اضغط **"إضافة"**

**الخيار الثاني: رفع ملف**
1. اختر **"رفع ملف"** (Upload File)
2. حمّل `index.html`
3. وافق على الإعدادات

#### 4. إعداد التقييم
- اذهب إلى **"الواجبات"** (Assignments)
- انقر **"إنشاء واجب جديد"** (Create Assignment)
- اربطها بالدورة
- اضبط الدرجات: **100 نقطة**

## ⚫ Blackboard

### الخطوات:

#### 1. الإعداد الأولي
1. **تسجيل الدخول** إلى Blackboard
2. اذهب إلى **غرفة الدورة** (Course Room)
3. اضغط **"وضع التحرير مشغل"** (Edit Mode)

#### 2. إضافة المحتوى
1. اختر **"المحتوى"** (Content)
2. اضغط **"إنشاء ملف"** (Create File) أو **"رفع ملف"** (Upload File)
3. حمّل ملف `index.html`
4. وافق على الإعدادات

#### 3. إعداد الاختبار
1. اذهب إلى **"الاختبارات"** (Tests)
2. اضغط **"إنشاء اختبار جديد"** (Create New Test)
3. أضف أسئلة على أساس محتوى الدورة
4. اضبط النقاط والدرجات

#### 4. تفعيل التتبع
- اذهب إلى **"المراقبة"** (Monitor)
- فعّل **"تتبع التقدم"** (Track Progress)
- عيّن **"علامات الإكمال"** (Completion Marks)

## 🌐 Google Classroom

### الخطوات:

#### 1. الإعداد
1. **تسجيل الدخول** إلى Google Classroom
2. اختر **الفصل**
3. اضغط **"عمل"** (Work)

#### 2. إضافة المادة التعليمية
1. اضغط **"إنشاء"** (Create) ثم **"مادة تعليمية"** (Material)
2. أعطِ العنوان: "درس الحوكمة الصحية"
3. أضف الوصف والتعليمات
4. أضف الرابط أو الملف
5. اضغط **"نشر"** (Post)

#### 3. إنشاء واجب/اختبار
1. اضغط **"إنشاء"** (Create) ثم **"واجب"** (Assignment)
2. أعطِ اسماً: "اختبار الحوكمة"
3. أضف التعليمات
4. اربط الدورة
5. عيّن درجة: **100 نقطة**
6. اضغط **"نشر"** (Post)

#### 4. المتابعة
- استخدم قسم **"الدرجات"** (Grades)
- راقب **"درجات الطلاب"** (Student Scores)

1. **تسجيل الدخول** إلى Schoology

## 📦 SCORM Packages

### إنشاء حزمة SCORM

#### الخطوة 1: التحضير
```bash
# بناء الدورة
npm run build

# نسخ الملفات المُترجمة
cp -r .output/public/* scorm-package/
```

#### الخطوة 2: إنشاء imsmanifest.xml
```xml
<?xml version="1.0" encoding="UTF-8"?>
<manifest version="1.1" 
  xmlns="http://www.imsproject.org/xsd/imscp_v1p1">
  
  <metadata>
    <lom>
      <general>
        <title>Healthcare Governance Models</title>
        <description>Interactive learning course</description>
        <language>ar</language>
      </general>
    </lom>
  </metadata>

  <organizations default="org1">
    <organization identifier="org1">
      <title>Governance Course</title>
      <item identifier="item1" identifierref="resource1">
        <title>Main Course</title>
      </item>
    </organization>
  </organizations>

  <resources>
    <resource identifier="resource1" type="webcontent">
      <file href="index.html"/>
    </resource>
  </resources>

</manifest>
```

#### الخطوة 3: ضغط الملفات
```bash
# استخدم برنامج الضغط
# قم بضغط المجلد بصيغة .zip
# تأكد من وجود imsmanifest.xml في الجذر
```

#### الخطوة 4: الرفع
- احفظ الملف باسم: `governance-course.zip`
- رفعه على منصة التعليم

```javascript
// مثال: إرسال حدث الإكمال
const statement = {

    score: {
      scaled: 0.85,
      raw: 85,
      min: 0,
      max: 100
    },
    success: true,
    completion: true
  }
}
```

## 🎯 نصائح عامة | General Tips

### ✅ قبل الرفع
- [ ] تأكد من عمل جميع الروابط
- [ ] اختبر الأيقونات والصور
- [ ] جرّب على أجهزة مختلفة
- [ ] اختبر على متصفحات متعددة

### ✅ إعدادات الدرجات الموصى بها
- **الدرجة الكاملة:** 100 نقطة
- **درجة النجاح:** 60 نقطة
- **محاولات مسموحة:** 3
- **عرض النتائج:** فوري

### ✅ إعدادات التتبع الموصى بة
- **تفعيل تتبع التقدم:** ✅
- **تتبع الأنشطة:** ✅
- **حفظ التقييمات:** ✅
- **توليد التقارير:** ✅

### ✅ اختبار ما بعد الرفع
1. ادخل كطالب جديد
2. أكمل جميع الشاشات
3. أجب على الاختبار
4. تحقق من الدرجة
5. اخرج وادخل من جديد (اختبر الحفظ)

### ⚠️ مشاكل شائعة وحلولها

| المشكلة | السبب | الحل |
|--------|------|------|
| الأيقونات لا تظهر | مشكلة CDN | تحمل Font Awesome محلياً |
| الأنشطة لا تتفاعل | مشكلة JavaScript | فعّل JavaScript في الإعدادات |
| المحفوظات لا تعمل | localStorage معطّل | استخدم sessionStorage بدلاً منه |
| الشاشة مشوهة | مشكلة CSS | امسح ذاكرة التخزين المؤقتة |

## 📞 الدعم الفني | Technical Support

### في حالة المشاكل:

1. **اختبر على متصفح آخر**
   ```
   Chrome → Firefox → Edge → Safari
   ```

2. **امسح ذاكرة التخزين المؤقتة**
   ```
   Ctrl+Shift+Delete (Windows/Linux)
   Cmd+Shift+Delete (Mac)
   ```

3. **تحقق من حجم الملف**
   ```
   يجب أن يكون أقل من 100 MB
   ```

4. **تحقق من الاتصال بالإنترنت**
   ```
   للتحميل الأول للأيقونات والخطوط
   ```
