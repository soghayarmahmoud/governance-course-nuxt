## 📤 رفع على LMS Platforms

### Moodle
1. اذهب إلى **محررات الدورة** > **أضف نشاط**
2. اختر **محتوى SCORM** أو **ملف HTML**
3. قم بإعدادات الدرجات: **100 نقطة**, **60 للنجاح**

### Canvas / Blackboard / Google Classroom
اتبع نفس الخطوات الأساسية:
1. أضف محتوى جديد
2. حمّل الملف أو الرابط
3. أعد الإعدادات

## 🎮 الاستخدام | Usage

### للطلاب
1. افتح الدورة من منصة التعليم
2. اضغط "ابدأ الرحلة التعليمية"
3. تنقل بين الشرائح باستخدام الأزرار
4. اضغط على البطاقات لكشف المعلومات
5. أجب على الاختبار لإكمال الدورة

### اختصارات لوحة المفاتيح
- `→` سهم اليسار: الشريحة التالية
- `←` سهم اليمين: الشريحة السابقة
- `Tab`: التنقل بين العناصر
- `Enter/Space`: تفعيل الأزرار
